# nanaapp

Мини-приложение для Telegram на базе React + Vite.

## Запуск

1. Установить Node.js 18+.
2. Выполнить `npm install`.
3. Запустить локально: `npm run dev`.
4. Собрать прод-версию: `npm run build`.

## Telegram Mini App

После деплоя на `https://` URL этот адрес можно подключить к боту в `@BotFather` как Mini App.
